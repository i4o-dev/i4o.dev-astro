# i4o.dev

-   Framework: [Astro](https://astro.build/)
-   Styling: [TailwindCSS](https://tailwindcss.com/)
-   Deployment: [Railway](https://vercel.com/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/i4o-dev/i4o.dev
  cd i4o.dev
  pnpm i
  pnpm dev
```
